## [Keyboard shortcuts](https://defkey.com/zathura-shortcuts)
- **D**: Toggle dual page view
- **A**: Fill width
- A number and then **Shift + G**: Go to page number
- **G** then **G**: Go to first page
- **Shift + G**: Go to last page
